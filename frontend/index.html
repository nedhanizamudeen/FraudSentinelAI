About FraudSentinel

FraudSentinel is an AI-based fraud detection system that analyzes transaction data and classifies each transaction as legitimate or fraudulent.

It uses an ensemble machine learning model combined with a risk adjustment layer to calculate fraud probability and generate a final decision.

The system supports both single transaction analysis and CSV batch analysis, providing outputs such as risk score, recommendation, explanation, counterfactual insights, and financial impact.

All results shown are generated in real time from the connected backend model.
